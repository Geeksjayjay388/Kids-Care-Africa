# Kids Care Africa - NGO Website

A responsive React.js website for a non-profit organization dedicated to helping children in Africa.

## Features

- **3 Main Pages**: Homepage, Donation Page, Blog Page
- **Responsive Design**: Mobile-first approach
- **Modern UI**: Clean design with Tailwind CSS
- **Interactive Elements**: Hover effects, animations
- **Payment Integration Ready**: Structured donation form

## Pages Overview

### 1. Homepage (`/`)
- Hero section with call-to-action buttons
- "How You Can Help" section
- Project showcase (Water, Shelter, Food, Education)
- Impact statistics

### 2. Donation Page (`/donate`)
- Custom donation amount selection
- Project-specific donations
- Multiple payment methods
- Real-time impact calculator
- Secure payment assurance

### 3. Blog Page (`/blog`)
- Stories and updates
- Category filtering
- Newsletter subscription
- Social sharing

## Tech Stack

- **React 18** - Frontend library
- **React Router DOM** - Page routing
- **Tailwind CSS** - Styling framework
- **Lucide React** - Icon library
- **Vite** - Build tool (optional)
